# 基本概念
- **两点之间**的超文本传输协议(**双向协议**)
    - 两点之间：不限于<服务器--主机>，也可以<服务器--服务器>
    - 超文本：HTML就是典型的超文本，本身是文字文件，内部通过标签定义了图片、视频等的链接，经过浏览器的解释，呈现出有文字和画面的网页
